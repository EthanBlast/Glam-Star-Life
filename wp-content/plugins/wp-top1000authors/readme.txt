== Description ==
Display the Top 1000 WordPress Plugin Authors.

Based on the idea of W-Shadow.com Top 1000 Plugin Authors.

See Demo:
http://www.daveligthart.com/top-1000-wordpress-plugin-authors/

== Installation ==

1. Extract zip in the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Requires wp-content/cache to be writable by webserver.
4. Call the php function wpt1000_display(); in your theme.

You're all set!

Note: If you really can't get the plugin to work; i'll help you out!
Just post a mail to: dligthart@gmail.com?subject=wp-top1000authors
